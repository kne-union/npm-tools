# Prompts 文档索引

本项目包含多个前端开发辅助提示词文档，用于指导 AI 完成组件国际化、业务模块创建等常见任务。

## 文档列表

### 1. 业务组件国际化

**功能**: 指导完成 React 组件的国际化改造

**适用场景**: 
- 组件需要支持多语言切换
- 为现有组件添加国际化功能
- 创建中英文语言包

**核心内容**:
- 创建 withLocale.js HOC 包裹组件
- 创建中英文语言包文件
- useIntl Hook 使用方法
- 不同组件类型的国际化模式（主组件、FormInner、getColumns、Action）
- createWithRemoteLoader 组件的特殊处理方式

**使用方式**: 阅读 [原文档](./prompts/业务组件国际化.md) 了解详情

---

### 2. 创建业务模块

**功能**: 基于 BizUnit 架构模式生成完整的前端业务模块

**适用场景**: 
- 快速创建标准的 CRUD 业务模块
- 需要列表展示、筛选、表单操作等功能
- 生成符合项目规范的业务组件结构

**核心内容**:
- BizUnit 核心组件配置与使用
- 列配置（getColumns）定义
- 表单组件（FormInner）实现
- 详情页（Detail）开发
- 操作按钮配置
- 筛选条件配置
- 文档示例规范

**使用方式**: 阅读 [原文档](./prompts/创建业务模块.md) 了解详情
